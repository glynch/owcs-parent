# Oracle WebCenter Sites Bill of Materials

This project is to manage the versions of OWCS dependencies.

See [Oracle WebCenter Sites 12.2.1.3.0](https://docs.oracle.com/en/middleware/webcenter/sites/12.2.1.3/index.html)

## Installation

```bash
git clone https://github.com/glynch/owcs-parent.git
cd owcs-parent
./install.sh <JSK_HOME>
./mvnw clean install
```
